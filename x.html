Q3.

撰寫一個程式，使用者可以輸入攝氏溫度(資料型別為int)後，會顯示攝氏溫度(佔4個字元)，和華氏度數(整數顯示4位小數3為，空白補0)。 

# input Celsius and Fahrenheit
Celsius = int(input("請輸入攝氏溫度: "))
print(Celsius)

# compute
Fahrenheit = Celsius * 9/5 + 32
print(Fahrenheit)

參考解答

Celsius = int(input("請輸入攝氏溫度: "))
print(Celsius)

# compute
Fahrenheit = Celsius * 9/5 + 32
print('%08.3f' % Fahrenheit)

錯誤分析

  • 未達到格式要求：題目要求整數部分顯示4位且小數部分顯示3位數，不足的位數用空白補充。
  • 解決方法：應使用格式化字串來正確顯示華氏溫度。
