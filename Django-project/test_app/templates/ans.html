Q3.

撰寫一個程式，讓使用者輸入攝氏溫度（int）。顯示攝氏溫度（佔4個字元）和華氏溫度（顯示整數部分4位，小數部分3位，用空白補齊）。

程式碼範例


# 輸入攝氏溫度
Celsius = int(input("請輸入攝氏溫度: "))
print(f"{Celsius:>4}")  # 攝氏溫度佔4個字元

# 計算華氏溫度
Fahrenheit = Celsius * 9 / 5 + 32
print(f"{Fahrenheit:>7.3f}")  # 顯示為4位整數和3位小數，補齊空白

參考解答


Celsius = int(input("請輸入攝氏溫度: "))
print(f"{Celsius:>4}")

# 計算華氏溫度
Fahrenheit = Celsius * 9 / 5 + 32
print(f"{Fahrenheit:>7.3f}")

錯誤分析

  • 未達到格式要求：題目要求整數部分顯示4位且小數部分顯示3位數，不足的位數用空白補充。
  • 解決方法：使用格式化字串來正確顯示華氏溫度，例如 `f"{Fahrenheit:>7.3f}"`。
