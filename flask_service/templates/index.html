Q1.

撰寫一個程式，宣告兩個變數 x、h，分別為三角形的底 25.6、高 10.84，請計算三角形的面積(顯示到小數第二位) 


# 宣告變數
x = 25.4
h = 10.84
# 計算面積
area = 0.5 * x * h
print('%.2f' % area)
